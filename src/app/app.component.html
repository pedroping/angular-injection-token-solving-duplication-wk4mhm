Angular Injection Token Demo

  • We have two modules ProductModule & SomeOtherModule
  • ProductModule Provides ProductService using the PRODUCTSERVICE instance injection Token
  • SomeOtherModule Provides SomeService using the PRODUCTSERVICE instance of injection Token
  • AppComponent uses the injection token PRODUCTSERVICE inject the productService from ProductModule/ProductService
  • We import both Modules in the appModule
  • The Order of import does not matter. AppComponent will always get the PRODUCTSERVICE from ProductModule/ProductService
